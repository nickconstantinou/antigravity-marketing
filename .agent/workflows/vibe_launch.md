# Vibe Launch Protocol

## 1. Research Phase
1.  **Command:** `@research [topic]`
2.  **Process:** Use `deep_research.md` (Workflow) to find the "Bleeding Neck" problem.
3.  **Check:** Does `research/[topic]_brief.md` exist?

## 2. Strategy Phase
1.  **Command:** `@positioning`
2.  **Process:** Use `positioning.md` (Skill 02) to define the Angle.
3.  **Command:** `@creative`
4.  **Process:** Use `creative-strategist.md` (Skill 11) to define the Vibe (Colors, Fonts).

## 3. Execution Phase (The Build)
1.  **Command:** `@visuals`
2.  **Process:** Use `visual-producer.md` (Skill 12) to generate assets based on the Vibe.
3.  **Command:** `@copy-direct`
4.  **Process:** Use `direct-response.md` to write the headlines using the Positioning Angle.
5.  **Command:** `@architect`
6.  **Process:** Scaffold the page.

## 4. Review Phase
1.  **Command:** `@audit`
2.  **Process:** Use `vibe-critic.md` to run the "2AM Bar Test" from `brand-voice.md`.
3.  **Refine:** Iterate until Vibe Score > 90.

## 5. Deployment & Distribution
1.  **Manual:** Deploy to Vercel/Netlify.
2.  **Command:** `@atomize`
3.  **Process:** Use `content-atomizer.md` to turn the landing page into:
    *   5 Tweets (Thread).
    *   2 LinkedIn Posts.
    *   1 Newsletter (`newsletter.md`).
    *   1 Email Sequence (`email-sequences.md`).
