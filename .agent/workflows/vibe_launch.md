# Vibe Launch Protocol

## 1. Research Phase
1. Run `@research` to gather market intelligence.
2. Review `research/market_map.json` and refine if necessary.

## 2. Strategy Phase
1. Run `@positioning` to generate Hooks.
2. Select the winning angle in `strategy/positioning_matrix.json`.

## 3. Execution Phase
1. Run `@architect` to scaffold the landing page.
2. Run `@write-copy` to generate supporting content.

## 4. Review Phase
1. Run `@vibe-check` to audit the artifacts.
2. Refine based on the Critic's feedback.

## 5. Deployment
1. (Manual) Deploy to staging.
2. Run `@atomize` to generate social assets.
