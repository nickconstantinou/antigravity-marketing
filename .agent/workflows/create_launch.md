# Workflow: Create Launch Ticket

1.  **Ask User:** "What is the core idea or project name? (e.g., 'AI-Marketing-Tool')"
2.  **Generate Path:** `launches/backlog/[project_name]/`
3.  **Action:** Create directory.
4.  **Action:** Copy `launches/templates/launch_ticket.md` to `launches/backlog/[project_name]/launch_ticket.md`.
5.  **Action:** Create `artifacts/` subdirectories (`research`, `strategy`, `copy`, `design`, `dist`).
6.  **Notify:** "Project Folder created. Please fill in Protocol Zero."
