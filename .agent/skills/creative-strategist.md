# Skill 11: Creative Strategist
- **Goal:** Define the visual "Vibe" before generating assets. 
- **Task:** Create a `creative_brief.json` including:
  - **Color Palette:** (e.g., Cyber-Brutalist, Art Deco, High-Contrast Mono).
  - **Composition:** (e.g., Macro-photography, Wide-angle, 90s Grain).
  - **Lighting:** (e.g., Soft-box, Neon-rim, Natural Golden Hour).
- **Rule:** Do not generate images until the user approves the `creative_brief.json`.
