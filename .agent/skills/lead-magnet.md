# Skill 04: High-Value Lead Magnets
- **Goal:** Create "Value-First" opt-ins.
- **Formats:** Checklists, 'Look-under-the-hood' Loom scripts, Cheat Sheets.
- **Validation:** If the lead magnet doesn't solve a problem found in `keyword-research.md` in < 5 minutes, reject the draft.
- **Directives:** No 50-page ebooks. Give them a "Quick Win" in < 5 minutes.
