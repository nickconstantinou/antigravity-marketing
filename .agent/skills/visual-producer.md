# Skill 12: Image & Product Photography
- **Objective:** Generate specialized assets (Product Shots, Social Graphics, Hero Images).
- **Protocol:**
  1. Call `glif_run` or `replicate_run` using the specific model defined in `creative_brief.json`.
  2. For Social Graphics: Optimize for the platform (1080x1350 for IG, 1200x630 for X).
  3. For Video: Generate 5-second "Product Reveal" animations via Runway/Luma MCP.
